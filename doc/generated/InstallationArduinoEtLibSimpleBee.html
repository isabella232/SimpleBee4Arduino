Requirements

  • IDE Arduino 1.6
  • une carte Arduino Uno
  • Arduino Wireless Proto Shield
  • un module XBee S1 de digi http://www.digi.com/products/wireless-wired-embedded-solutions/zigbee-rf-modules/xctu

Tlchargement et installation de IDE

Aller sur le site de Arduino et tlcharger la dernire version en fonction de votre OS.

  • Mac OS X

Tlchargement et installation de la bibliothque SimpleBee4Arduino

Tlcharger la dernire version de la bibliothque SimpleBeeMessenger (zip file) https://github.com/Orange-OpenSource/SimpleBeeMessenger/releases.

Dans l'IDE Arduino IDE, aller croquis > Importer bibliothque > ajouter bibliothque et slectionner le fichier zip tlcharg.

Dans croquis > Importer bibliothque, vous avez maintenant la bibliothque SBMessenger.

Tlcharger la dernitre version de la bibliothque SimpleBee4Arduino (zip file) https://github.com/Orange-OpenSource/SimpleBee4Arduino/releases.

Dans l'IDE Arduino IDE, aller croquis > Importer bibliothque > ajouter bibliothque et slectionner le fichier zip tlcharg.

Dans croquis > Importer bibliothque, vous avez maintenant la bibliothque SimpleBee4Arduino.
